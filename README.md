# 🌧️ Rainfall Impact on Agricultural Productivity in India (1966–2017)
### Interactive Power BI Dashboard

## 📌 Project Overview
This project analyzes the historical impact of the South-West Monsoon (June–September) on the agricultural productivity of India across 18 states over a 51-year period. By transforming raw meteorological and agricultural data into an interactive Power BI dashboard, this project identifies critical trends, geographical disparities, and the direct correlation between rainfall volume and crop yield (specifically Rice and Wheat).

## 🛠️ Tools & Technologies Used
*   **Microsoft Power BI:** Data Modeling, DAX, Data Visualization, Interactive Dashboards
*   **Power Query Editor:** Data Cleaning, Transformation, and Structuring
*   **DAX (Data Analysis Expressions):** Custom measures for Time Intelligence, YoY Growth, and Efficiency Indexes

## 📊 Dashboard Architecture (4 Pages)
1.  **Executive Overview:** High-level KPIs and dual-axis correlation charts.
2.  **Monsoon Patterns:** Historical rainfall distribution, moving averages, and filled maps.
3.  **Crop Yield Analysis:** Deep dive into yield vs. rainfall correlation via Scatter plots.
4.  **State-Level Deep Dive:** Heatmaps and Rainfall Efficiency Indexes to evaluate regional agricultural resilience.

## 💡 Key Insights
*   **Technological Resilience:** While major droughts (e.g., 2002) still cause visible yield drops, overall rice productivity has steadily tripled since 1966 due to modern farming techniques (Green Revolution), reducing absolute monsoon dependency.
*   **Geographical Disparities:** Coastal and Eastern states (Kerala, West Bengal) receive over 5x more Jun-Sep rainfall than Northern and Southern states (Tamil Nadu).
*   **The Efficiency Paradox:** States like Punjab and Tamil Nadu produce the highest yields per millimeter of monsoon rain. This indicates heavy reliance on advanced irrigation networks and alternative weather systems (NE Monsoon) rather than the primary summer monsoon.
